xtb_mctc_strings::writenum Interface Reference

Generic interface for writing a number to a string. The number is left justified in the string. The calling syntax is 'call writenum(number,string,format)' where 'number' is a real number or an integer, 'string' is a character string containing the result, and 'format' is the format desired, e.g., 'e15.6' or 'i5'. Detailed Description

Generic interface for writing a number to a string. The number is left justified in the string. The calling syntax is 'call writenum(number,string,format)' where 'number' is a real number or an integer, 'string' is a character string containing the result, and 'format' is the format desired, e.g., 'e15.6' or 'i5'.
