Detailed Description

Only used for writing Python extension modules. Datastructure to hold table of methods of your Python extension module. Put exactly one instance at Fortran module level.

Python 3: initialise and configure in PyInit_*module name* function with bind(c, name="PyInit_*module_name*") attribute and type(c_ptr) return value. Python 2: initialise in init*module name* subroutine with bind(c) attribute

Pass the configured PythonMethodTable to PythonModuleinit
